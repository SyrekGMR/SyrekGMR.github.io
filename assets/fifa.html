Visualising Data From FIFA20

Fifa 20 offers a great source of data from the football world. Each player within the game possesses a set of attributes aimed at indicating their proficiency in the various aspects of football, such as dribbling. These combine to provide an overall rating of each player, with the best players in the world typically assigned a rating of over 90. This project looks to explore the correlation between these, allowing an interesting and unique insight into the world of football as modelled by Fifa 20.

Below are presented a series of interactive visualisations which can be manipulated by the user, allowing you to change the attribute on each axis as well as the player's nationality. The first two plots display the overall rating distribution over the player age and the counts of the preferred foot. The last plot can be modified using the two drop-down menus below, allowing for the attribute plotted along each axis to be manually set.

This webpage is hosted on Github Pages and hence may experience slowdowns, please allow a few seconds for the SVG plots to re-render each time.
